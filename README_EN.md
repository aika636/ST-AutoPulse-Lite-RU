Расширение для SillyTavern, позволяющее персонажам **активно** отправлять вам сообщения.

> **Примечание для международного сообщества:**  
> Это расширение в первую очередь разрабатывается для китайскоязычного сообщества. Английская документация предоставляется «как есть», чтобы эти возможности стали доступны всем. Поддержка обращений на английском может быть отложена или недоступна. Вы можете форкнуть репозиторий или отправлять PR, если хотите помочь с локализацией или исправлением ошибок!

## ✨ Возможности

- **Запланированные автоматические сообщения**: установите интервал (1–180 минут), и персонаж будет сам начинать разговор.
- **Запланированные задачи**: настройте ежедневные, еженедельные или одноразовые сообщения (например, «Говорить «Доброе утро» каждый день в 9:00»).
- **Фоновая обработка**: даже при закрытом браузере серверный таймер Node.js продолжает работать. Пропущенные сообщения ставятся в очередь и появятся при возвращении.
- **Уведомления на рабочем столе**: нативные всплывающие уведомления, когда персонаж обращается к вам.
- **Пользовательские промпты**: полный контроль над стилем, контекстом и поведением активных сообщений.
- **Slash-команды**: быстрое управление расширением через `/autopulse on|off|trigger|status|<минуты>`.

### 🆕 Новые возможности V2

- **Система эмоционального давления** 🧠: чем дольше персонаж ждёт ответа, тем выше его «давление». Сообщения становятся чаще, тон — тревожнее. Когда вы наконец отвечаете, персонаж реагирует (облегчение, недовольство, гнев) в зависимости от того, как долго его игнорировали.
- **Система всплывающих уведомлений ревности** 💢: при переключении с персонажа А на персонажа Б, персонаж А может с настраиваемой вероятностью отправить всплывающее уведомление с ревностью или протестом.
- **Фоновая активность без обрывов**: использует Web Worker вместо простых интервалов, чтобы браузер не замораживал расширение, когда вкладка неактивна.
- **Совместимость API**: автоматически подстраивается под старую и новую версии API `generateQuietPrompt` в SillyTavern.

## 📦 Установка

> **Расширение состоит из двух частей**: UI Extension (фронтенд) + Server Plugin (бэкенд).
> Встроенный установщик расширений SillyTavern устанавливает только фронтенд. Серверный плагин необходимо устанавливать **вручную**.

### Шаг 1: Установка UI Extension (автоматически)

В SillyTavern откройте **Extensions → Install Extension** и вставьте ссылку на этот репозиторий:

```
https://github.com/NANA3333333/ST-AutoPulse
```

### Шаг 2: Установка Server Plugin (вручную — обязательно)

> ⚠️ **Этот шаг обязателен**, иначе таймеры остановятся при закрытии браузера, а **система эмоционального давления НЕ будет работать**.

1. Перейдите в корневой каталог установки SillyTavern.
2. Найдите или создайте папку `plugins/`.
3. Внутри `plugins/` создайте папку с именем `autopulse/`.
4. Скопируйте файл `server-plugin/index.js` из этого репозитория в созданную папку. *(При обновлении с V1 просто перезапишите старый файл).*

Итоговая структура:

```
SillyTavern/
└── plugins/
    └── autopulse/
        └── index.js
```

### Шаг 3: Включение серверных плагинов

Отредактируйте `config.yaml` в корневом каталоге SillyTavern и убедитесь, что в нём есть следующий блок:

```yaml
enableServerPlugins: true
```

### Шаг 4: Перезапустите сервер SillyTavern

**Важно:** перезапустите Node.js-бэкенд (командную строку / терминал, в котором запущен SillyTavern). Обновления веб-страницы недостаточно.
После перезапуска откройте панель Extensions. Вы должны увидеть «ST-AutoPulse» с зелёным индикатором «Connected to Server».

## 🚀 Использование

1. Откройте панель **Extensions** в левой части SillyTavern.
2. Найдите раздел настроек **ST-AutoPulse**.
3. Отметьте «Enable Auto Messages».
4. Установите желаемый интервал сообщений.
5. (Опционально) Настройте промпты для сообщений и задач.
6. (Опционально) Включите систему эмоционального давления и систему ревности.

### Система эмоционального давления

При включении персонаж испытывает тем более сильную тревогу, чем дольше вы не отвечаете:

| Уровень давления | Эффект |
|-----------------|--------|
| 😊 Уровень 0 | Нормальный интервал сообщений. |
| 😐 Уровень 1 | Интервал сокращён на 30 %. «Уже скучаю». |
| 😟 Уровень 2 | Интервал сокращён на 50 %. «Начинаю волноваться». |
| 😰 Уровень 3 | Интервал сокращён на 70 %. «Тревожно». |
| 😭 Уровень 4 | Интервал сокращён на 80 %. «Эмоциональный предел». |

**Реакция на возвращение**: когда вы наконец отправляете сообщение, персонаж реагирует на игнорирование (облегчение, гнев, ласка). Реакция происходит один раз, после чего давление сбрасывается до 0.

### Система ревности

Если вы переключаетесь с персонажа А на персонажа Б, персонаж А может через некоторое время отправить всплывающее сообщение с жалобой на то, что его игнорируют ради другого. Вы можете настроить:

- Вероятность срабатывания (0–100 %)
- Минимальную и максимальную задержку (в секундах)
