# История изменений

## v2.0.0 (2026-02-21)

### 🆕 Новые возможности

#### Система эмоционального давления 🧠

- Чем дольше персонаж ждёт вашего ответа, тем выше уровень давления (0–4).
- Чем выше давление, тем короче интервал между сообщениями (1.0x → 0.2x), в промпт добавляется тревожность.
- **Реакция на возвращение**: после вашего ответа персонаж реагирует в зависимости от времени ожидания:
  - Низкое давление → радость, успокоение 😊
  - Среднее давление → лёгкое недовольство, капризы 😐
  - Высокое давление → неудовлетворённость, гнев 😡
- Реакция на возвращение срабатывает только один раз, после чего давление сбрасывается до нуля.
- Все параметры настраиваются пользователем (максимальный уровень, включение / отключение реакции).
- Встроенные тестовые кнопки: давление +1, тестовый запуск, тест возвращения.

#### Плавающее окно ревности 💢

- При переключении чата оставленный персонаж может отправить уведомление с ревностью.
- Уведомление в виде плавающего окна с эффектом стекла в правом нижнем углу: аватарка + сообщение ревности.
- **Выбор персонажей**: можно указать, какие персонажи будут ревновать (не выбрано — не срабатывает).
- **Пользовательский промпт**: можно написать собственный промпт для управления стилем вывода.
- Настраиваемые параметры: вероятность срабатывания, минимальная / максимальная задержка.
- Автоматическая очистка от цепочек рассуждений, описаний действий и прочего мусора.
- Встроенная тестовая кнопка.

### 🐛 Исправление ошибок

#### Исправление фонового опроса

- Для опроса используется Web Worker вместо `setInterval`.
- Расширение больше не замораживается в фоновых вкладках браузера.
- При сбое Worker автоматически переключается на `setInterval`.

#### Совместимость с API SillyTavern

- Добавлен слой совместимости `callGenerateQuietPrompt`.
- Автоматическая адаптация под новую (объектные параметры) и старую (строковые параметры) версии API.
- Поддержка SillyTavern 1.12.0 ~ 1.16.0+.
